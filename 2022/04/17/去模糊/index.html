第四组 图像去模糊--De-blurring

发表于2022-04-17更新于2022-04-20

题目介绍

  • 从模糊的图像中重建一个清晰的图像需要改变低频和高频信息。虽然传统的ResBlock可以很好的捕捉图像的高频分量，但它往往会忽略低频信息所以提出了一种带有卷积块(ResFFT-Conv块)的残差快速傅里叶变换，能够捕获长期和短期交互，同时集成低频和高频残差信息。

项目进度

第一周:

  • 学习深度学习pytorch实践，安装相关软件

    第二周:

  • 学习深度学习pytorch实践，熟悉软件的使用和环境配置

    第三周:

  • 阅读论文《Deep Residual Fourier Transformation for Single Image Deblurring》
  • 论文链接：https://arxiv.org/abs/2111.11745

    第四周

  • 学习了解快速傅里叶变换（FFT）的算法原理及代码解释
  • 文章链接：http://www.360doc.com/content/18/0424/21/42387867_748463569.shtml

    第五周

  • 阅读多篇基于深度学习的图像去模糊的论文，了解一种基于数据驱动的判别先验的盲图像去模糊方法、一种基于生成对抗网络的单幅图像去模糊算法、一个基于加权空间金字塔特征融合网络的图像去模糊方法和一个基于卷积神经网络(CNN)的图像去模糊方法。

    第六周

  • 运行代码，对代码进行初步注释，但尚未跑通

    第七周

  • 读代码，学习代码中的EBlock,DBlock,AFF,SCM,FAM等模块，并进一步完善对代码的注释
  • 完整代码链接：[INVOKERer/DeepRFT: The code for ‘Deep Residual Fourier Transformation for Single Image Deblurring’ (github.com)](INVOKERer/DeepRFT: The code for ‘Deep Residual Fourier Transformation for Single Image Deblurring’ (github.com))
