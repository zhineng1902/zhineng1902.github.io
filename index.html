第一组 生产对抗网络--Generative Adversarial

发表于2022-04-17更新于2022-04-20

题目介绍

  • 神经网络容易学习捷径——它们经常对简单的相关性进行建模，而忽略了可能更好地泛化起来的更复杂的相关性。先前关于图像分类的研究表明，深度分类器不是学习与物体形状的联系，而是倾向于利用与低级纹理或背景的虚假相关性来解决分类任务。在这项工作中，我们朝着更健壮和可解释的分类器迈出了一步，这些分类器明确地公开了任务的因果结构。基于目前在深度生成建模方面的进展，我们建议将图像生成过程分解为独立的因果机制，这些机制是我们在没有直接监督的情况下训练的。通过利用适当的归纳偏差，这些机制解开了物体形状，物体纹理和背景;因此，它们允许生成反事实图像。我们展示了我们的模型在MNIST和ImageNet上生成此类图像的能力。此外，我们表明，尽管是合成的，但反事实图像可以提高分布外的鲁棒性，同时在原始分类任务上性能略有下降。最后，我们的生成模型可以在单个 GPU 上进行有效训练，利用常见的预训练模型作为归纳偏差。

第二组 知识蒸馏--Distilling Knowledge in Neural Network

发表于2022-04-17更新于2022-04-20

题目介绍

  • 在深度学习的背景下，为了达到更好的预测效果，高性能的神经网络往往层数纵深，参数密集，难以部署在资源受限的设备上。知识蒸馏作为一种模型压缩方法，可以将复杂网络(称为教师网络)学到的知识传递给小网络(称为学生网络)，使学生网络达到较好的预测效果，同时学生网络所占计算参数数量及内存大小远小于教师网络，从而使其移植到低内存低算力的终端上成为了可能。本文介绍了知识蒸馏的理论知识和作用机制，并基于 flower 数据集利用知识蒸馏方法将 GoogLeNet（教师网络，大小为 50MB）进行压缩，“知识”选择教师网络的输出层，实验结表明，蒸馏后的学生网络预测准确性要比单独训练的小模型高，且其模型大小不足 100KB，模型压缩效果显著

第三组 对比学习--Contrastive Learning

发表于2022-04-17更新于2022-04-21

题目介绍

  • 对比式学习着重于学习同类实例之间的共同特征，区分非同类实例之间的不同之处。与生成式学习比较，对比式学习不需要关注实例上繁琐的细节，只需要在抽象语义级别的特征空间上学会对数据的区分即可，因此模型以及其优化变得更加简单，且泛化能力更强。
  • 我们主要是研究原型对比学习，是将对比学习与聚类联系起来的无监督表示学习方法

第五组 时间序列异常检测--Time-series anomaly detection

发表于2022-04-17更新于2022-04-20

题目介绍

  • 现实世界的系统会有大量的连续数据，这些时序数据的异常检测对于保障系统安全、避免经济损失有着重要意义，例如服务器、地空设备的监测等。
    时序数据的异常检测已经受到了广泛关注。但是经典异常检测方法（例如LOF、OC-SVM）很少考虑时间信息，这使得他们在时序场景中应用受限。基于深度网络方法，往往是通过循环网络（RNN）学习时序数据点级别的表征，进而依靠重建或预测误差进行判定，然而，此类方法所学习的点级别表征信息量较小，并且可能被正常模式主导，使得异常点难以区分。
    因此，如何获取更具信息含量的表征，进而定义更加具有区分性的判据对于时序异常检测尤为关键。所以，将重点放在无监督条件下的时间序列异常检测上。

项目进度

第一周:

  • 1、先大致了解了：
    （1）异常一般分为：
    点异常，连续性异常（群组异常）上下文一场（序列异常）
    （2）异常检测的解决方法按模型类别分可以都包括：有监督（Supervised）、半监督（Semi-supervised）、无监督（Unsupervised）、复合模型方法（Hybrid）。其中有监督不作为重点研究方向的原因：异常类别多种多样，很难去遍历或枚举出异常原因。获取异常的样例比较少。所以把重点放在无监督的方法上。
    （3）比较常见的时序异常检测方法：
    基于统计（某时刻值偏离某段时间的均值或方差值超过定义的阈值）
    Autoregressive Moving Average(AMRA),异常未来时间的值，如果偏离正常值一定阈值，认定异常
    One-Class SVM(根据正常点判断新的事件是否为异常)
    KNN（近邻的样本数量，或近邻的样本）
    Auto Encoder或者PCA降维（根据重构误差来计算是否异常，假设异常在重构时会丢失信息）

    第二周:

  • 1、通过上周的论文学习得知：监督异常检测中的方法主要可归为两类：
    侧重于通过精心设计的循环网络学习 pointwise representation，并通过重建或自回归任务进行自监督，因此一个自然而实用的异常标准是时间点的重构或预测误差。但是由于异常的罕见性，时间点的表示对于复杂的时间模式信息较少，并且由于正常时间点的主导作用，使得异常难以区分。
    通过显式关系建模方法（explicit association modeling），向量自回归和状态空间模型属于这一类。例如采用gnn来构建这种关联性，虽然gnn具有更强的表现能力，但学习到的图仍然仅限于单个时间点，这对于复杂的时间模式是不够的。此外，基于子序列的方法通过计算子序列之间的相似度来检测异常，但在探索更广泛的时间上下文信息时，这些方法无法捕获每个时间点与整个系列之间的细粒度时间关联（只能捕获子序列级别粗粒度的关联）。
    所以学习的论文将Transformer引入无监督时序数据异常检测任务。首先提出series-association，这一个特性是直接可以通过Transformer模型来获取，每个时间点的时间关联可以从自注意力图（self-attention map）中获得，它表示了关联权重在时间维度上所有时间点的分布。 每个时间点的关联分布可以为时间上下文提供更丰富的信息描述来描述动态模式，例如时间序列的周期或趋势。

  • 异常检测是自主监视系统的一个关键目标，应该能够提醒异常观察。在本项目中，我们使用深度神经网络的整体异常检测系统，使用Unet模型利用pytorch框架完成系统需要的检测无人机图像中的各种异常。异常包括漏油、破土动工、材料储存和未经授权的倾倒。由于电机数量大、范围广，人工检测漏油等故障会耗费大量人力、 物力 、财力。所以本项目研究使用无人机来代替工人检测，大大节省检测时间， 及时修补 ，提高工作效率。
