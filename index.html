第五组 时间序列异常检测--Time-series anomaly detection

发表于2022-04-17更新于2022-04-19

题目介绍

  • 现实世界中的系统总是以一种连续的方式工作，可以产生由多个传感器监测的若干连续的测量结果，如工业设备、空间探测器等。从大规模的系统监控数据中发现故障，可以减少到从时间序列中发现异常时间点，这对于保证安全，避免经济损失具有重要意义。但是异常通常很少见，并且被巨大的法线点所掩盖，这使得数据标记困难且昂贵。因此，我们将重点放在无监督环境下的时间序列异常检测上。

第三组 对比学习--Contrastive Learning

发表于2022-04-17更新于2022-04-19

题目介绍

  • 对比式学习着重于学习同类实例之间的共同特征，区分非同类实例之间的不同之处。与生成式学习比较，对比式学习不需要关注实例上繁琐的细节，只需要在抽象语义级别的特征空间上学会对数据的区分即可，因此模型以及其优化变得更加简单，且泛化能力更强。
  • 我们主要是研究原型对比学习，是将对比学习与聚类联系起来的无监督表示学习方法

第二组 知识蒸馏--Distilling Knowledge in Neural Network

发表于2022-04-17更新于2022-04-19

题目介绍

  • 在深度学习的背景下，为了达到更好的预测效果，高性能的神经网络往往层数纵深，参数密集，难以部署在资源受限的设备上。知识蒸馏作为一种模型压缩方法，可以将复杂网络(称为教师网络)学到的知识传递给小网络(称为学生网络)，使学生网络达到较好的预测效果，同时学生网络所占计算参数数量及内存大小远小于教师网络，从而使其移植到低内存低算力的终端上成为了可能。本文介绍了知识蒸馏的理论知识和作用机制，并基于 flower 数据集利用知识蒸馏方法将 GoogLeNet（教师网络，大小为 50MB）进行压缩，“知识”选择教师网络的输出层，实验结表明，蒸馏后的学生网络预测准确性要比单独训练的小模型高，且其模型大小不足 100KB，模型压缩效果显著

第一组 生产对抗网络--Generative Adversarial

发表于2022-04-17更新于2022-04-19

题目介绍

  • 神经网络容易学习捷径——它们经常对简单的相关性进行建模，而忽略了可能更好地泛化起来的更复杂的相关性。先前关于图像分类的研究表明，深度分类器不是学习与物体形状的联系，而是倾向于利用与低级纹理或背景的虚假相关性来解决分类任务。在这项工作中，我们朝着更健壮和可解释的分类器迈出了一步，这些分类器明确地公开了任务的因果结构。基于目前在深度生成建模方面的进展，我们建议将图像生成过程分解为独立的因果机制，这些机制是我们在没有直接监督的情况下训练的。通过利用适当的归纳偏差，这些机制解开了物体形状，物体纹理和背景;因此，它们允许生成反事实图像。我们展示了我们的模型在MNIST和ImageNet上生成此类图像的能力。此外，我们表明，尽管是合成的，但反事实图像可以提高分布外的鲁棒性，同时在原始分类任务上性能略有下降。最后，我们的生成模型可以在单个 GPU 上进行有效训练，利用常见的预训练模型作为归纳偏差。
